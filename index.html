Run Trackerbuddy on your computer

📦 One‑time setup

git clone https://github.com/billgoldbergmania/fitness-tracker.git
cd trackerbuddy
npm install
npm run dev

Then open http://localhost:3000 – your data lives in tracker.db.

🌐 On your local network

npm run dev -- -H 0.0.0.0

Access from any device on the same Wi‑Fi using your computer’s IP (e.g. http://192.168.1.10:3000).

📦 Production build

npm run build
npm run start

Frequently asked questions

❓ Will my data be uploaded anywhere?

Never. The app only uses a local SQLite file (tracker.db). No external API calls, no analytics, no tracking.

❓ Can I use it on my phone?

Yes – run the server on your computer and open the network URL on your phone. The interface is responsive.

❓ How do I switch between users?

Go to Settings → User Management. Add, rename, or delete users. Click “Switch” to change active user.

❓ Can I export my data?

Yes – use the “Export as CSV” button in Settings. All weight logs and workout sets will be downloaded.

❓ What if I lose the database file?

Back up tracker.db regularly. If deleted, the app recreates a fresh database with default data (old logs are lost).

❓ Does it work with pounds / inches?

Absolutely – choose your units in Settings. All charts and logs automatically convert.
